Установка

После распаковки скачанного архива требуется зайти в папку со скриптами и ввести команды:

pip install -r requirements.txt

sudo apt-get install smartmontools

sudo python3 login.py

При запуске появится окно входа и после авторизации будет запущен скрипт отправляющий информацию о состоянии вашего накопителя на данный веб ресурс. Информацию о состоянии вашего накопителя вы можете проверить на вкладке "Ваши устройства".
